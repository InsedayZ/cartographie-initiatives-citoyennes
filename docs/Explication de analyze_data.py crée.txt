Le fichier analyze_data.py que j'ai créé sert à faire l'analyse exploratoire initiale des données CSV. Voici son rôle précis :
🔍 Objectif Principal
Comprendre la structure et le contenu du dataset avant de créer les visualisations dans l'application Streamlit.
📊 Analyses Effectuées
Informations générales : Nombre de lignes/colonnes, noms des variables
Types de données : Identification des variables numériques/catégorielles
Valeurs manquantes : Vérification de la qualité des données
Statistiques descriptives : Min, max, moyenne pour les variables numériques
Distributions catégorielles : Comptage des modalités (types d'initiatives, villes, etc.)
Analyse géographique : Vérification des coordonnées latitude/longitude
🎯 Utilité Concrète
Phase de découverte : Première exploration du dataset inconnu
Validation des données : S'assurer qu'il n'y a pas d'erreurs ou d'incohérences
Planification des visualisations : Identifier quels graphiques créer
Génération d'insights : Découvrir les patterns intéressants à approfondir
📈 Résultats Obtenus
Ce script m'a permis de découvrir que :
100 initiatives sur 6 types différents
Aucune donnée manquante (dataset propre)
Répartition géographique sur 10 villes françaises
Période 2020-2024 avec croissance continue
Variables d'impact social/environnemental bien structurées
En résumé : C'est l'étape d'analyse préliminaire indispensable avant de construire l'application Streamlit finale !
